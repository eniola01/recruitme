In this tutorial we use a plain text editor (like Notepad) to edit HTML.I believe this is the best way to learn HTML.However, professional web developers often prefer HTML editors like FrontPage or Dreamweaver, instead of writing plain text
Follow the steps below to create your first html web page.

Step 1: Open Notepad (PC)
windows 8 or later: Open the start screen (the window symbol at the bottom left on your screen). Type notepad.
window 7 or earlier:
Open start > All Programs > Accessories > Notepad


Step 1: Open TextEdit (MAC)
Open finder > Application > Text-edit
Also change some preferences to get the application to save the html file correctly. In the prefrences > Format > choose "Plain Text"
Then under "Open and save", Check the box that says "Display HTML files as HTML code instead of formatted text"
The open a new document to place the code

Step 2: write some HTML code
Let write some HTML codes

      <html>
              <head>
                  <title>Basic page</title>
               </head>
              <body>
                  <h1>My First Heading</h1>
                  <p>My first paragraph.</p>
              </body>
          </html>

Step 3: Save the HTML page
Save the file on the text editor. Select File > Save as in the Notepad Menu.
Name the file "index.htm" and set the encoding to "UTF-8" (Which is the preferred encoding for HTML files).

Note: You can save your html file using ".htm" or ".html" as the file extension. There is no diffrence

Step 4: View the HTML page in your browser
Open the file save as index (right click and choose open with, select the browser of your choice i.e Firefox and click on it)
